# mdprint

Print markdown in python

## Installation

Clone the repo:

```sh
git clone https://github.com/JeffTheK/mdprint
```

Then run `make install` in the project directory

```sh
make install
```

## Usage

Just run mdprint in terminal with specified file

```sh
mdprint some_file.md
```

## Building

```sh
make install
```

### Tests

```sh
make test
```
